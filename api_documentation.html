Postman Usage Guide
1
Open Postman and click Environments. Create a new environment and add a variable base_url set to http://localhost:5000. Use {{base_url}} in all your request URLs.
2
For all POST / PATCH / DELETE requests that send data in body, go to Headers tab and add: Content-Type = application/json
3
To add request data, go to Body tab → select raw → switch format to JSON. Then type your JSON like: { "email": "test@mail.com", "password": "123456" }
4
For file uploads (Knowledge Base), go to Body → select form-data. Set key type to File for the files field and select your document.
5
For routes with :id in URL, replace it with the actual MongoDB ID. Example: {{base_url}}/api/history/65a4bc...
6
For protected admin routes, after login copy the token from the response. In Postman, go to Authorization tab → select Bearer Token → paste the token.

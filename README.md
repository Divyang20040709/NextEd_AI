# 🚀 NextEd AI

NextEd AI is an intelligent AI-powered education platform designed to enhance learning experiences using modern AI technologies. It enables users to interact with AI for academic assistance, document understanding, and smart insights generation.

---

## 📌 Project Overview

NextEd AI aims to bridge the gap between traditional learning and modern AI-driven education by providing:

* 📄 Document understanding (PDF / image / text)
* 🤖 AI chatbot for academic queries
* 🧠 Smart insights & explanations
* 🔄 Multi-input processing (text, images, etc.)
* ⚡ Fast and user-friendly interface

This project is built as a **final year engineering project** focusing on real-world AI applications.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Divyang20040709/NextEd_AI.git
cd NextEd_AI
```

---

### 2️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
